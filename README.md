Smart Utility CLI App (Python)

This project is a beginner-friendly Python CLI application that combines multiple utility features into a single interactive program.

📌 Overview

This is a menu-driven Command Line Interface (CLI) application built in Python.
It allows users to perform multiple basic utilities within a single interactive program.

The application runs in a continuous loop, displays a menu of options, and executes different functionalities based on user input.

🚀 Features
🔢 Basic Calculator
Addition, subtraction, multiplication, and division
Handles division by zero
🔍 Even / Odd Checker
Determines whether a number is even or odd
🎓 Grade Checker
Assigns grades based on marks
Includes input validation
🔐 Login System
Simple authentication using predefined credentials
❌ Exit Option
Allows user to safely terminate the program
🛠️ Tech Stack
Python (Core concepts: loops, conditionals, input/output, functions)
🧠 Concepts Practiced
while loops for continuous execution
Conditional statements (if, elif, else)
Logical and comparison operators
Input validation
Basic program structuring
▶️ How to Run
Make sure Python is installed
Save the file as cli_app.py
Run the program:
python cli_app.py
